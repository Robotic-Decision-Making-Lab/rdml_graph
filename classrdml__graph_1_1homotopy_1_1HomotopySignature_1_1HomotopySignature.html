Detailed Description

A basic structure to handle HSignatures These are implemented as the homotopy signature given in: S.

Bhattacharya, M. Likhachev, V. Kumar (2012) Topological constraints in search-based robot path planning

Note these are homotopy invariants instead of homology. This implies order matters. IMPORTANT: Feature numbers must start at 1 and not 0. Yes this is weird, but makes computation and storage signicantly faster.
