AStar A generic implementation of the AStar algorithm.

An optimal graph search algorithm. If looking for shortest path to all nodes, see Dijkstra's algorithm. REQUIRED

Parameters
start- the start state of the search OPTIONAL Functions g and h have input types of (state, data, goal)
g- a goal function to determine if the passed, state is in the goal set.
h- a heuristic function for the AStar search needs type (state, data, goal)
data- a potential set of input data for huerestics and goal states.
goal- a potential set of goal data (REQUIRED by default)
out_tree- if true, output tree, otherwise do not output a tree.
keepEdges- [opt] if true, keep the edges in the path.
keepNodes- [opt] if true, keep the nodes in the path.
Returns
- list, cost an optimal list states to the goal state. - if no path return empty list and infinte cost. [first state, —, goal state]

BFS Breadth First Search algorithm.

This has an optional budget which restricts expansion beyond the budget given the cost of the state function returned REQUIRED

Parameters
start- the starting OPTIONAL
budget- the budget of the search, if ignored, no cost budget is given.
g- a goal condition that must be met in order to be in the returned states. g(n, data, goal)
data- input data for the goal function if required.
goal- any goal data required by g function.
keepEdges- [opt] if true, keep the edges in the path.
keepNodes- [opt] if true, keep the nodes in the path.
Returns
- list of structs of [(path, cost),...] or [([n1,n2,n3,...], cost), ...]

dijkstra's algorithm (All nodes) This is dijkstra's algorithm ran to find the shortest path to all reachable nodes of a graph from the start location.

See any algorithms book for description of dijkstra's algorithm. Very similar to the above AStar algorithm without being single query, and without a huerestic function.

Parameters

Returns
- a dictionary of every SearchState in the tree. (key is state)
