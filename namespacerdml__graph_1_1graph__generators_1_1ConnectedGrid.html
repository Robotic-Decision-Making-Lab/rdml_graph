connected_grid Generate a graph as a connected 4 or 8 grid.

Can define size of the grid given max sizes and ticks of the underlying field.

Parameters
map- map with options {'bounding', 'obs', 'hazards'}
x_ticks- array like of N ticks
y_ticks- array like of M ticks
sampleF- the sampling function for the PRM sampleF(map, num_samples)
collision- the collision function to check for connection between nodes: collision(parent, child, map) parent->child nodes, map is the given map of the PRM.
connection- the connection function setting costs and exact method to calculate. connection(parent, child, map, cost=None) - connects the parent to the child node.
grid_size- [opt] the size of the grid given the x and y ticks.
conn_8- [opt] true 8-connected grid (diagonals), false, 4-connected grid
bidirectional- [opt] sets if the PRM is guarenteed to be bidirectional and if bidirectional collisions and costs need to be checked.
Returns
a list of nodes with grid conencted edges
